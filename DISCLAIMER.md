# Disclaimer

## Clinical Use

This repository documents a research project. **It is NOT intended for clinical use, medical diagnosis, or patient care decisions.** The system described here was deployed under supervised academic conditions at KTÜ Farabi Hospital with appropriate ethics board approval and physician oversight.

Any attempt to use the methodologies described here for actual patient diagnosis must:
- Obtain independent ethics board approval
- Be supervised by qualified medical professionals
- Comply with all applicable medical device regulations (e.g., MDR in EU, FDA in US, TITUBB in Turkey)
- Undergo proper clinical validation

## Data

- **No patient data** is included in this repository
- **No model weights** trained on clinical data are included
- **No ethically-approved medical images** are included
- All result tables contain only aggregate statistical metrics

The clinical dataset used in the original study was obtained from KTÜ Farabi Hospital under ethics board approval and remains exclusively with the institution.
